用户注册

请求URL

/user/join POST

请求参数 application/x-www-form-urlencoded

参数名 类型 必须 描述
token string 否 用于注册认证的token，获取验证码时返回

请求体 application/json

{
	"mail":"string",
	"password":"string",
	"nickname":"string //用户昵称",
	"isManager":"boolean //是否为管理员",
	"verifyCode":"string //验证码"
}

返回结果

{
	"code":"string",
	"msg":"string",
	"data":"string"
}
